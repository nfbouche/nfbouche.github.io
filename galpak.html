GalPaK3D

A tool for modeling galaxy kinematics

GalPaK3D is a Python package for the kinematic modeling of galaxies. It uses a Bayesian approach to fit 3D kinematic data cubes with a simple rotating disk model, providing robust estimates of the galaxy's rotation curve, inclination, and other parameters.

Key features:

  • Bayesian inference using MCMC
  • Handles various data formats (FITS, etc.)
  • Outputs rotation curves, velocity maps, and more

For more information, visit the doc GalPaK3D documentation and checkout the code at gitlab repository, also available on PyPI.
